inserts formatted block of links in the post ex: [smartlink id="1"]

== Description ==

Operation:
The plugin enables the bar of the post formatting options icon
Clicking the icon will open a window that contains the form for registration of links
Links will be saved in the bank and will be available in Smart Links List tab
Sign In / Choose a link unit the following code is inserted in the post where XXX is the id of the block saved in the database.

Example: the following links on this page are being entered via the plugin is not formatted but I'm working on the option to insert CSS for block

[smartlinks id="3"]

== Installation ==

1. install the plugin from the wordpress repository version (always updated)
2. after downloading the plugin activate it
3. click on new post or new page
4. to enter the link unit click the yellow icon in the edit bar
5. window that appears enter the links in the form and click insert
6. finished the plugin created a short tag eg [SmartLinks id = "1"]
7. save the post / page and see the result in your blog
